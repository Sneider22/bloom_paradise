# 🏕️ Camp Loyaltown - Admin System

Un panel administrativo responsivo tipo Single Page Application (SPA) para la gestión y control de los campistas de **Camp Loyaltown**.

## 📌 Propósito
Facilitar el registro, seguimiento y análisis de la información de los campistas en una interfaz moderna, limpia y completamente adaptada tanto a dispositivos móviles como de escritorio.

## ⚙️ Funcionalidad
- **Autenticación:** Sistema básico de inicio y cierre de sesión con persistencia.
- **Control de Campistas (CRUD):** Registro completo para agregar, listar, actualizar y eliminar expedientes de campistas.
- **Dashboard Estadístico:** Gráficos dinámicos e indicadores clave sobre la ocupación y distribución del campamento.
- **Diseño Responsivo:** Panel adaptado con menús interactivos colapsables (sidebar drawer) para dispositivos móviles.
- **Persistencia Local:** Almacenamiento local automático de la información a través del navegador.

## 🛠️ Tecnologías Utilizadas
- **Core:** React 19 & Vite
- **Estilos y Diseño:** Tailwind CSS v4 & Lucide React (Iconos)
- **Gráficos:** Recharts
- **Persistencia:** LocalStorage API
